# Version 1 Project Summary

## Project Title
Smart Servo IoT Controller - Basic Version

## Description
This version uses an ESP32 microcontroller to host a simple web interface that controls a servo motor. The HTML interface is written directly inside the Arduino code, making it simple and lightweight.

## Objective
The aim of this version was to learn how a microcontroller can connect to WiFi, host a basic web server, and interact with external hardware in real time.

## System Flow
1. ESP32 connects to WiFi.
2. A web server runs on the ESP32.
3. The user enters the ESP32 IP address into a browser.
4. The browser displays lock and unlock buttons.
5. The ESP32 receives the request and moves the servo motor.

## Learning Outcomes
- Arduino C++ programming
- Servo motor control
- ESP32 WiFi setup
- Basic web server routing
- Hardware-software integration
