# Smart Servo IoT 

## Overview
This repository contains a WiFi-enabled smart servo control system developed as a personal learning project during my free time. The project explores embedded systems, IoT communication, web-based control, and hardware-software integration using Arduino/ESP32.

The repository is split into two versions:

## Version 1: Basic Web Interface
This version reflects the original style of the project. The web interface is simple and embedded directly inside the Arduino firmware. It allows users to control a servo motor through basic browser buttons.

## Version 2: Enhanced Scalable Interface
This version improves the project structure by separating the frontend files from the firmware. It includes a cleaner dashboard interface, slider control, preset buttons, status display, and a more scalable project architecture.

## Technologies Used
- ESP32 / Arduino
- Arduino C++
- HTML
- CSS
- JavaScript
- WiFi Web Server
- Servo Motor Control
- IoT Communication

## Hardware Used
- ESP32 development board
- SG90 servo motor or equivalent
- Breadboard
- Jumper wires
- USB cable
- Optional external 5V power supply
